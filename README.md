# VLA API Test

电脑屏幕理解 VLM 测试脚本，用于测试 VLA 模型的屏幕描述、UI 元素识别和 GUI 操作推理能力。

## 功能

- **纯文本对话测试** - 验证模型基础对话能力
- **屏幕截图描述** - 分析电脑屏幕内容，描述窗口、应用、UI 元素
- **屏幕操作问答** - 针对 GUI 操作进行问答（可交互元素识别、点击位置建议等）
- **多屏幕状态对比** - 检测界面变化（可选，需要 test1.png 和 test2.png）

## 环境要求

- Python 3.8+
- openai
- Pillow
- vLLM 服务运行在 `http://localhost:8000/v1`

安装依赖：

```bash
pip install openai pillow
```

## 使用方法

### 1. 启动 vLLM 服务

```bash
vllm serve <model_name> --dtype half
```

### 2. 运行测试

```bash
python VLAapi_test.py
```

脚本会自动检查 `test.png` 是否存在，如不存在会提示创建示例图像。

### 测试图像

- 主测试图像：`test.png`
- 多图测试（可选）：`test1.png`, `test2.png`

## 配置

如需修改 API 地址，编辑脚本第 15-18 行：

```python
client = openai.OpenAI(
    base_url="http://localhost:8000/v1",
    api_key="EMPTY"
)
```

## 适用模型

- Qwen2-VL
- UI-TARS
- 其他支持 OpenAI API 格式的 VLM 模型
